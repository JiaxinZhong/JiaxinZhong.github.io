Grammar & Language

Math Symbols & Equations

  1. A space should be placed between the numerical value and the unit symbol. For example, 1.2 dB is correct, while 1.2dB is not. If you are writing in LaTeX, I recommend you use the siunitx by \usepackage{siunitx}. Then the command \SI{<value>}{<unit>} automatically print the space between the numerical value and the unit symbol. See Sec. 7.2 in NIST Guide for the Use of the International System of Units (SI) for more information.
  2. Physical units must not be italicized. For example, 1.2 dB is correct, while 1.2 dB is not.
  3. Round up to only one decimal place for decibels. For example, 1.2 dB is correct, while 1.23 dB is not.
